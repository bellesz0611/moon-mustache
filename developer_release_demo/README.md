# Developer Release Demo

This package demonstrates Moon Mustache in a developer-facing release workflow.

It focuses on an SDK / package release pack:

- generate a GitHub release body
- generate an install-and-upgrade snippet
- generate a migration summary
- validate the bundle and print a generation plan first

Run it with:

```bash
moon run developer_release_demo
```

This helps show that Moon Mustache can support developer-ecosystem publishing tasks in addition to operations, content, and starter-repository generation.
