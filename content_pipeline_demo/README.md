# Content Pipeline Demo

This package is a third independent consumer-style example for Moon Mustache.

It demonstrates a content-oriented workflow:

- Markdown overview generation
- announcement text generation
- HTML card fragment generation
- profile-aware bundle rendering
- validation and bundle-plan reporting

Run it with:

```bash
moon run content_pipeline_demo
```

Together with `downstream_consumer/` and `adoption_demo/`, this helps show that Moon Mustache can support multiple downstream domains instead of only one narrow showcase.
