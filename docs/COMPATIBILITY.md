# Compatibility Matrix

This document tracks the current compatibility posture of Moon Mustache against common Mustache expectations.

Current status summary:

- 47 focused project spec cases organized into 9 suites
- 194 imported core and optional `mustache/spec` fixture cases, all passing with zero skips
- about `8032` handwritten effective MoonBit lines; imported generated fixtures are disclosed separately
- Markdown compatibility report generated by `moon run spec_report`
- official fixture compatibility report generated by `moon run official_spec_report`
- scenario-level workflow report generated by `moon run scenario_report`
- validation and plan reports available for bundle manifests
- 85 automated tests passing locally
- 2048 deterministic differential cases passing against `mustache.js`
- dedicated Linux and Windows workflows validate both the browser ESM and API bridge

Exact current repository counts are tracked in `docs/METRICS_SNAPSHOT.md`.
