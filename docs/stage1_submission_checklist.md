# 第一阶段提交清单（历史归档）

- 本文件记录第一阶段准备过程，当前专项奖材料请以 [专项鼓励奖证据索引](SPECIAL_AWARD_EVIDENCE.md) 为准。
- 把 [stage1_application.md](stage1_application.md) 里的姓名、联系方式、GitHub、GitLink 链接补全
- 在 GitHub 和 Gitlink 创建同名公开仓库并保持同步
- 补足不少于 10-20 次有效提交，不要用空提交或无意义拆分
- 准备一个简洁 README，至少写清楚项目目标、计划功能和当前进度
- 建好基础目录，例如 `src/`、`test/`、`examples/`
- 放一个最小可运行示例，证明项目已启动
- 如果你决定不用 `Moon Mustache` 这个方向，只改项目名、简介、核心功能和参考项目即可，其余结构可以直接复用
