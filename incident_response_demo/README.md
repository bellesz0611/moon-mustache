# Incident Response Demo

This package demonstrates Moon Mustache in an incident-response workflow.

It focuses on an operational handoff scenario:

- generate a Markdown incident runbook
- generate a status-page update draft
- generate an escalation-contact environment file
- validate the bundle and print a generation plan first

Run it with:

```bash
moon run incident_response_demo
```

This helps show that Moon Mustache is useful not only for scaffolding and content generation, but also for operational response kits with multi-file outputs.
