# Starter Repo Demo

This package demonstrates Moon Mustache as the core of a starter-repository generator.

It focuses on a practical bootstrapping scenario:

- generate a `README.md`
- generate a `moon.mod`
- generate a `.gitignore`
- generate a starter `src/main.mbt`
- validate the bundle and print a generation plan first

Run it with:

```bash
moon run starter_repo_demo
```

Together with the other consumer demos, this makes the repository look less like one library with one example and more like a package already capable of supporting multiple downstream workflow styles.
